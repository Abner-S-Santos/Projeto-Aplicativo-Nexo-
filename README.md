# Projeto-Aplicativo-Nexo-
Nexo é uma plataforma em formato de fórum voltada à inclusão e à troca de experiências, com foco em pessoas com necessidades especiais, além de oferecer um espaço separado para discussões diversas.  O projeto foi idealizado com o objetivo de criar um ambiente seguro, organizado e acolhedor, promovendo conexão, empatia e acessibilidade digital.
